# BingKmlParser
Parser for kml data to be used with [Bing Maps API](https://www.microsoft.com/maps/)

* Supported features:
    * Pushpin
    * Polyline
    * Polygon
    * MultiGeometry
    * Style
    * StyleMap
    * External Style
    * GroundOverlay (experimental)
* [Usage example](src/index.html)
